# Release v0.2.2

## Highlights

- Added GOFRUXA account-based cloud configuration.
- Added Baidu APP ID / Key sync from website account.
- Added NVIDIA API Key sync for AI translation.
- Added automatic browser-side credential import after extension login.
- Added domain-level translation settings.
- Improved Chrome / Edge / Android Kiwi compatibility.

## Installation

Download `fanyi_chrome.zip` from this release.

- Chrome / Edge: unzip and load as an unpacked extension.
- Kiwi Browser: install the zip from `chrome://extensions/`.

See `README.md` for detailed instructions.
